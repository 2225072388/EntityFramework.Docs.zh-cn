# <a name="ef-core-tools--extensions"></a>EF Core 工具和扩展

工具和扩展为 Entity Framework Core 提供了额外功能。

扩展由多种源生成。 并非所有扩展均作为 Entity Framework Core 项目的组成部分进行维护。 考虑使用第三方扩展时，请务必评估质量、授权、支持等因素，确保其满足要求。
