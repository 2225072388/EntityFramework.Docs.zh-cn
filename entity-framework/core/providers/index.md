<span data-ttu-id="b18cb-222">安装后，需采用 `OnConfiguring` 方法或 `AddDbContext` 方法（如果使用的是依赖关系注入容器）在 `DbContext` 中配置提供程序。</span><span class="sxs-lookup"><span data-stu-id="b18cb-222">Once installed, you will configure the provider in your `DbContext`, either in the `OnConfiguring` method or in the `AddDbContext` method if you are using a dependency injection container.</span></span> <span data-ttu-id="b18cb-223">例如，以下行使用传递的连接字符串配置 SQL Server 提供程序：</span><span class="sxs-lookup"><span data-stu-id="b18cb-223">For example, the following line configures the SQL Server provider with the passed connection string:</span></span>
