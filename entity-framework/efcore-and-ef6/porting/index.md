# <a name="porting-from-ef6-to-ef-core"></a>从 EF6 移植到 EF Core

由于 EF Core 有一些根本性变化，我们不建议尝试将 EF6 应用程序迁移至 EF Core，除非你有令人信服的理由进行此项更改。 你应该将从 EF6 移至 EF Core 视作移植而不是升级。
